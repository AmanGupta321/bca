ANIMATION INDUSTRY

  In Latin 'Anima' means soul. Animation is all about giving soul to character. It is about moving something which cannot move itself.

  Some of the popular animation techniques are :
  • 2D Cell Animation
    • Also known as traditional animation, 2D animation involves the creation of high volume of separate drawings that defines a sequence.

  • 3D CGI Animation
    • This techniques makes extensive use of animation software programmes, 3D objects are constructed using curves or 2D geometric figures.

  • 3D Motion Capture Animation
    • This process of creating 3D characters is similar to the 3D CGI animation technique : however,the techniques differ with respect to the time when the animation effects are introduced.
